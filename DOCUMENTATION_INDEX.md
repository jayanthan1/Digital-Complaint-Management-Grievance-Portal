## üìù Quick Command Reference

### Backend Setup
```bash
cd backend
npm install
cp .env.example .env
# Edit .env with MySQL credentials
mysql -u root -p < src/config/database.sql
npm run dev
```

### Frontend Setup
```bash
cd frontend
npm install
npm start
```

### Access Application
- Frontend: http://localhost:4200
- Backend API: http://localhost:5000/api
